# Recrutări Festivaluri - Cashless Payment Systems

Platformă de recrutare pentru casieri la festivaluri (Beach Please, Untold, Kapital).

## Setup

```bash
npm install
npm run dev
```

## Deploy pe Vercel

1. Push pe GitHub
2. Conectează repo-ul la Vercel
3. Deploy automat

## Configurare

API-ul Google Apps Script este configurat în `app/page.js` (constanta `API_URL`).

## Structură

- `/` — Landing page + Formular + Status check (Beach Please 2026)
- `/ture` — Portal "Turele mele" (vizualizare program)
